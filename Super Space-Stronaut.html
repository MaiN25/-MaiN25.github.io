Super Space-Stronaut

About the game

I created a 2D Platformer project - Super Space-Stronaut while participating in the MSU Game Design and Development with Unity 2020 Specialization. I developed this game using the assets and guidance given during the lectures.

The modifications I added or chnage in the game are the following

  • Added two more levels.
  • Modified the level made during the lectures.
  • Added new enemies (flying enemies).
  • Added another requirement for winning (the need to defeat a certain amount of enemies).
  • Added a score increment each time an enemy is killed.

Game Instructions

  • The player controls would be both the keyboard arrows and WSAD.
  • The player would jump once you press the ENTER key.
  • To kill the enemies, you should jump on their heads.
  • In order to pass the new levels (2&3): you need to kill four enemies AND pick up the door key to win.
