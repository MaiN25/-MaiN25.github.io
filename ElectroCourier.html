ElectroCourier

About the game:

This game is a final group project for one of the courses I participated in at UT Austin (AET 334F). I was in charge of the UI/UX design & development and was part of the programming team. My responsibilities in this project were:

  • UI design & programming
  • Implemented the different UI elements & menus.
  • Player Animation (including implementation of the art one of my teammates created into unity)
  • Worked on the different enemies’ functionality along with another teammate.
  • Game management
  • Design the various pickup elements.
  • Input system
